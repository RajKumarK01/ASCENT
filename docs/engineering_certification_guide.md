## Role-to-certification mapping

**Cloud Engineer**
- Primary: AZ-204 (Developing Solutions for Azure)
- Secondary / advancement: AZ-305 (Azure Solutions Architect)

**DevOps Engineer**
- Primary: AZ-400 (DevOps Engineer Expert)

**Data Engineer**
- Primary: DP-203 (Data Engineering on Azure)

**Security Engineer**
- Primary: SC-200 (Security Operations Analyst)

## Recommended study pattern

- 1–2 hours of focused study daily.
- Weekly assessment checkpoints.
- Target a 75% practice score before booking the exam.
- AZ-305 has a prerequisite expectation of AZ-204 or equivalent experience.

## Skill areas by certification

- **AZ-204**: API development, Azure Functions, storage. Recommended ~20 hours.
- **AZ-305**: architecture, governance, networking. Recommended ~30 hours.
- **AZ-400**: CI/CD, monitoring, GitHub Actions. Recommended ~25 hours.
- **DP-203**: data pipelines, storage, stream processing. Recommended ~22 hours.
- **SC-200**: threat detection, Microsoft Sentinel, Microsoft Defender. Recommended ~20 hours.
